## EasyWeChat 是什么？

EasyWeChat 是一个 [微信](http://www.wechat.com) 非官方 SDK。

## 安装

EasyWeChat 的安装非常简单，因为它是一个标准的 [Composer](https://getcomposer.org/) 包。

### 环境需求

- PHP >= 5.3
- Mcrypt 拓展
